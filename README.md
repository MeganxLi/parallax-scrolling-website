### Folder Description

```
.
├── public
│   ├── fonts                   # 字體
│   └── images                  # 頁面使用的圖像
├── src
│   ├── assets/svg              # 使用 icon
│   ├── components              # 共用文件
│   ├── constants               # 管理設定
│   ├── page/                   # 對應的頁面
│   ├── route                   # Route 設定
│   ├── styled/                 # linaria css 樣式
│   ├── styles                  # 基本 css 樣式
│   ├── types                   # TypeScript 型別宣告
│   └── utils                  # 共用函數
├── ...
```

## Building

### Install Packages

```
$ npm install
```

### Start Dev Server

```
$ npm run dev
```
